# Student Grade Calculator
The Student Grade Calculator is a web application built using HTML, CSS, and JavaScript. It allows users to calculate Percentage of marks.
# Technologies Used
* HTML: Markup language for structuring the web page and form elements.
* CSS: Styling language for designing the layout and appearance of the web page.
* JavaScript: Programming language for implementing interactive features and calculations.
* GitHub Pages: Hosting platform used to deploy the project and make it accessible online.
